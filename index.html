jor1k: OR1000 Javascript Emulator Running Linux

Specifications

  • 32-Bit OR1000 Emulator with MMU, TICK counter and PIC (Hack for TLB Hardware Refill)
  • 32 MB Ram
  • UART 16550
  • ocfb Framebuffer 320x240 32bpp
  • Terminal Emulator
  • 5 MB image running Linux 3.1 and buszbox 1.19 (and much more)
  • Only 2300 lines of Javascript code!
  • Tested under Firefox and Chrome

Explore the Emulator

First wait until the operating system finished booting. Then press enter.

Get information about the system

Information about the running system
can be achieved by the /proc filesystem.
cat /proc/cpuinfo
shows the details about the cpu.
cat /proc/meminfo
shows the details about the memory.
top shows all running processes. Stop this program by pressing q.

Fun with the framebuffer

First try to find out the details about the framebuffer by the software fbset
There is a graphic demo demo available called fbinfo. Close the program by pressing CTRL+c .
You can fill the buffer manually with random data by executing
dd if=/dev/urandom of=/dev/fb0

Play games

Games are available. Try

  • hanoi
  • bs
  • knight
Close the program by pressing CTRL+c

Screensaver worm

Enter worm
and watch a few worms travelling on your screen. Close the program by pressing CTRL+c

Use the arbitrary precision calculator

By executing bc -l you can open a calculator.
2^1000 to calculate a big power or
4.*a(1) to calculate Pi.
